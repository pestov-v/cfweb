# CFWeb - Corporate Finance Web Application Gallery

Комплексна система управління корпоративними фінансами з ролями користувачів, управлінням продуктами та функціями регуляторного дотримання.

## Особливості системи

- **Управління користувачами** - система ролей та користувачів з детальними налаштуваннями
- **Управління ролями** - гнучке налаштування прав доступу для різних категорій користувачів
- **Управління продуктами** - каталог фінансових продуктів з можливістю редагування
- **Регламенти** - система управління нормативними документами
- **Довідники** - стандартизація даних та термінології
- **Інтеграція з БАРС** - модуль інтеграції з банківською системою
- **Електронний архів** - система електронного документообігу
- **Анкета клієнта** - форма збору та обробки інформації про клієнтів
- **Управління заявками** - система обробки та відстеження заявок
- **Граматичні відмінки** - модуль для правильного відмінювання імен у документах

## Технології

- Сучасний веб-інтерфейс
- Адаптивний дизайн
- Інтеграція з банківськими системами
- Система управління документами
